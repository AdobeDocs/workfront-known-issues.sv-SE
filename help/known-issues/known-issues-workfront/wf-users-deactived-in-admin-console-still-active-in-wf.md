# Användare: Användare som tas bort från Adobe Admin Console är fortfarande aktiva i Workfront

När en administratör tar bort en användare i Adobe Admin Console tas den användaren inte bort från Workfront. Användaren är fortfarande synlig och aktiv i Workfront.

_Först rapporterad den 3 februari 2026._
