# Användare: &quot;[!UICONTROL Only Allow SAML 2.0 Authentication]&quot;-inställningen saknas när en användare kopieras

>[!NOTE]
>
>Problemet korrigerades den 21 juli 2022.

När en gruppadministratör kopierar en användare och avmarkerar alternativet[!UICONTROL Send an invite email to this person]&quot;,[!UICONTROL Only Allow SAML 2.0 Authentication]kryssrutan visas inte som förväntat. Detta kan inträffa även när alla åtkomstkrav och behörighetskrav för den här åtgärden uppfylls.

_Först rapporterad den 16 maj 2022._
