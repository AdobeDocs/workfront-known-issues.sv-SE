# Godkännanden: Godkännanden låses automatiskt

När en användare försöker fatta ett beslut om ett bevis låses godkännandet och användaren kan inte fatta något beslut. Detta kan inträffa även om det inte finns någon anledning att låsa beviset.

_Först rapporterad den 16 februari 2026._
