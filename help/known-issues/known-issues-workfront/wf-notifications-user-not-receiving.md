# Meddelanden: Användaren får inga förväntade meddelanden

När en händelse inträffar som bör skicka ett meddelande till en användare, får användaren inte meddelandet. Detta kan inträffa även om inställningen för användarmeddelanden tillåter att meddelandet skickas.

Det här problemet påverkar både meddelanden i programmet och e-postmeddelanden.

_Först rapporterad den 12 februari 2026._
